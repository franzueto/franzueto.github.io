# franzueto.github.io

Personal site for [franzueto.dev](https://franzueto.dev) — built as a single, dependency-free HTML page.

## Stack

- Vanilla HTML, CSS, and JavaScript
- [Tailwind CSS](https://tailwindcss.com) via CDN (no build step)
- [Inter](https://fonts.google.com/specimen/Inter) + [Material Symbols](https://fonts.google.com/icons) via Google Fonts

## Running locally

No build step required. Just open the file in a browser:

```bash
open index.html
```

## Deployment

Hosted on [GitHub Pages](https://pages.github.com) with a custom domain.
